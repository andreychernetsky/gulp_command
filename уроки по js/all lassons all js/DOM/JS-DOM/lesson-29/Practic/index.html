Урок 29. Разные полезные темы.

Работа с формами

1. Обратилмя к любой форме, например к форме с номером 0

2. Переберем все формы с помощью цикла

3. Форме с именем form1 поставим текст '+', а форме form2 '-'

Обращение к элементам форм

1. Обратимя к форме номер 1, а затем к ее элементу номер 0

2. Перебером форму с момощью цикла и получим форму с номером 1 и всеми элементми этой формы поставим value '+'

Обращение к элементам форм по имени

1. Обратимся к элементам форм по имени

Работа с селектами

1. Пусть в переменной select лежит выпадающий список, тогда как - select.value мы можем получить value выбранного пункта списка

2. Записшем в select.value значение атрибута value определенного тего option - он старнет выбранным

3. Обратимя к выбранному пункту через option

Обращение к тегам option

1. С помощью свойства options, последовательно выведем содержимое атрибутов value тегов option

2. Запишем в текст кождого тега option '+', если хто выбранный пункт. А если не выбран то '-'
